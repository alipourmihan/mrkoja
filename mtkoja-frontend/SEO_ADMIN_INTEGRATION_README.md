# راهنمای ادغام پنل مدیریت SEO

## 📋 خلاصه تغییرات

پنل مدیریت SEO کاملاً در سیستم ادمین موجود ادغام شده است و شامل تمام ویژگی‌های مورد نیاز می‌باشد.

## 🎯 ویژگی‌های اضافه شده

### 1. کامپوننت SEO Management
- **مسیر**: `src/pages/admin/components/SeoManagement.vue`
- **ویژگی‌ها**:
  - داشبورد SEO با آمار و نمودارها
  - مدیریت صفحات SEO (کسب‌وکارها، دسته‌بندی‌ها، صفحات سفارشی)
  - تولید خودکار SEO
  - اعتبارسنجی SEO
  - تولید Sitemap و robots.txt
  - گزارش‌های تفصیلی

### 2. منوی ادمین
- **مسیر**: `src/pages/admin/components/AdminSidebar.vue`
- **اضافه شده**: منوی "مدیریت SEO" با آیکون مناسب

### 3. ادغام در داشبورد
- **مسیر**: `src/pages/admin/AdminDashboard.vue`
- **اضافه شده**: بخش SEO Management در داشبورد اصلی

## 🚀 نحوه استفاده

### دسترسی به پنل SEO
1. وارد پنل ادمین شوید
2. از منوی سمت راست روی "مدیریت SEO" کلیک کنید
3. سه تب اصلی در دسترس است:
   - **داشبورد**: آمار و نمودارهای SEO
   - **صفحات**: مدیریت صفحات SEO
   - **گزارش‌ها**: گزارش‌های تفصیلی

### ویژگی‌های اصلی

#### داشبورد SEO
- نمایش آمار کلی کسب‌وکارها و دسته‌بندی‌ها
- نمودار وضعیت تکمیل SEO
- نمودار توزیع Meta Tags
- دکمه‌های تولید Sitemap و robots.txt

#### مدیریت صفحات
- فیلتر و جستجو در صفحات
- انتخاب انبوه صفحات
- تولید خودکار SEO
- اعتبارسنجی صفحات
- ویرایش اطلاعات SEO

#### گزارش‌ها
- صفحات بدون SEO
- خطاهای اعتبارسنجی
- آمار عملکرد

## 🔧 تنظیمات فنی

### API Endpoints مورد نیاز
```
GET /api/admin/seo/dashboard
GET /api/admin/seo/pages
POST /api/admin/seo/generate-sitemap
POST /api/admin/seo/generate-robots
POST /api/admin/seo/generate-bulk
POST /api/admin/seo/validate
GET /api/admin/seo/reports
```

### وابستگی‌ها
- Vue 3
- Chart.js
- Axios
- Tailwind CSS
- Font Awesome
